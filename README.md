NCC Companion is a web platform designed to help students understand how to join NCC and support existing cadets with organized resources. Many students miss the chance to join NCC due to lack of awareness and unclear procedures. This solution centralizes all NCC-related information in one accessible place.


 **Features**

- Step-by-step guidance on how to join NCC
- FAQs and required documents for enrollment
- Senior cadet connect section for mentorship
- Secure login for NCC cadets using Firebase Authentication
- Cadet dashboard with parade info, uniform guide & reference notes
- AI-assisted NCC Guide chatbot for doubt clarification


** Tech Stack**

| Component | Technology |
|---------|------------|
| Frontend | HTML, CSS, JavaScript |
| Authentication | Firebase Authentication |
| Hosting | Firebase Hosting |
| AI/Chatbot | Google Gemini (planned / fallback logic implemented)|



# Problem We Solve

Students often miss NCC opportunities due to low awareness, unclear enrollment steps and lack of a central information source. Existing cadets also rely on scattered messages for updates. NCC Companion fixes this by providing a dedicated platform for aspirants & cadets.
